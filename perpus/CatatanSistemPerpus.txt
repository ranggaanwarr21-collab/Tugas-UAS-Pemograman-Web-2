1. Deskripsi Umum
Proyek ini adalah aplikasi sistem manajemen perpustakaan berbasis web untuk admin.
Fungsinya meliputi:
- Mengelola data buku
- Mengelola anggota
- Mencatat transaksi peminjaman dan pengembalian
- Melihat laporan aktivitas perpustakaan

Dibangun menggunakan:
- PHP (Prosedural)
- MySQL (XAMPP)
- Bootstrap (Tampilan sederhana)
- Metode koneksi: mysqli (terpisah di file includes/db.php)



2. Struktur Folder

perpus/
    actions/          Berisi proses backend (tambah, edit, hapus, login)
        buku_add.php
        buku_edit.php
        buku_delete.php
        anggota_add.php
        anggota_edit.php
        anggota_delete.php
        pinjam_add.php
        pinjam_kembali.php
        login.php

    includes/         File pendukung dan koneksi database
        db.php
        header.php
        footer.php
        functions.php

    pages/            Halaman utama untuk admin
        dashboard.php
        buku.php
        anggota.php
        peminjaman.php
        laporan.php

    sql/
        schema.sql    Struktur database MySQL

    index.php         Halaman login utama
    logout.php        Menghapus sesi admin
    style.css         (Opsional) gaya tambahan


3. Penjelasan Fungsi Folder & File

includes/
- db.php  
Menyediakan koneksi ke database perpus dengan variabel $conn.
- header.php / footer.php  
Elemen layout utama HTML (navigasi dan tampilan konsisten).
- functions.php  
Berisi fungsi bantu seperti format tanggal, validasi, dan pesan alert.

actions/
Berisi semua file proses CRUD (Create, Read, Update, Delete):
- buku_add.php / buku_edit.php / buku_delete.php  
Mengelola tabel buku.
- anggota_add.php / anggota_edit.php / anggota_delete.php 
Mengelola tabel anggota.
- pinjam_add.php 
Membuat transaksi peminjaman baru.
- pinjam_kembali.php 
Memperbarui status peminjaman menjadi dikembalikan.
- login.php 
Mengecek email & password admin untuk akses sistem.

pages/
Menampilkan data dari database:
- dashboard.php 
Menampilkan ringkasan data (jumlah buku, anggota, pinjaman).
- buku.php 
Tabel daftar buku + tombol tambah/edit/hapus.
- anggota.php 
Tabel daftar anggota perpustakaan.
- peminjaman.php 
Daftar transaksi peminjaman.
- laporan.php 
Ringkasan transaksi peminjaman/pengembalian.



4. Struktur Database (schema.sql)

Database: perpus
Tabel utama:
- admin 
Data login admin.
- buku 
Informasi buku (judul, pengarang, stok).
- anggota 
Data anggota perpustakaan.
- peminjaman 
Data transaksi peminjaman buku.

Contoh data awal:
Admin: 
Email: admin@perpus.com
Password: admin123



5. Mekanisme Login
1. Admin masuk lewat index.php.
2. Sistem memverifikasi data dari tabel admin.
3. Jika benar 
diarahkan ke pages/dashboard.php.
4. Jika salah 
pesan error muncul.
5. Logout menghapus session (logout.php).



6. Alur Umum Aplikasi
1. Admin login.
2. Melihat dashboard (statistik buku, anggota, pinjaman).
3. Menambah data buku atau anggota.
4. Mencatat peminjaman/pengembalian.
5. Melihat laporan aktivitas.



7. Pengembangan Lanjutan
Fitur yang dapat ditambahkan:
- Reset password via email.
- Filter laporan berdasarkan tanggal.
- Sistem multi-user (petugas perpustakaan).
- Tampilan responsif (Bootstrap 5).
